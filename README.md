# ESTS 2026 雅典年會・學習筆記

個人參加第 34 屆歐洲胸腔外科醫學會（ESTS）年會（2026 年 6 月・雅典）的學習筆記，將部分研究整理成「一個臨床問題、一篇文章」的形式，作為自我學習回顧與紀錄。

## 線上閱讀
開啟 `index.html`（若已啟用 GitHub Pages，可直接以網頁瀏覽）。

## 內容
- 依主題分類：早期肺癌切除範圍與淋巴結、手術入路與術式（微創）、轉移性疾病的外科角色、進展期 NSCLC 的多模態治療、高齡與共病的手術評估、縱膈腔與胸腺、全球調查與趨勢。
- 每篇結構：問題式標題 → 一句話結論 → 研究設計與數據 → 解讀與限制 → 出處。
